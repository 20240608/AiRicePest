## 📍 页面导航流程

```
主页 (/home)
  ├─ 知识库区域展示
  │   ├─ "换一批" → 刷新显示
  │   ├─ "查看全部" → 列表页
  │   └─ 点击卡片 → 详情页
  │
  └─ 侧边栏"打开知识库" → 展开/收起区域

知识库列表 (/knowledge)
  ├─ "主页"按钮 → 返回主页
  ├─ "返回"按钮 → 上一页
  ├─ 搜索和筛选
  └─ 点击卡片 → 详情页

详情页 (/knowledge/[id])
  ├─ "返回"按钮 → 上一页
  └─ 侧边栏导航 → 其他页面
```
